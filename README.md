# covid-tracker-vue

A small simple project build with [Vue.js](https://vuejs.org/) and [Tailwind CSS](https://tailwindcss.com/)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
